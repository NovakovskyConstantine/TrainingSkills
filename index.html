Git commands

  • git clone path to repository(скопировать файл из репозитория на жесткий диск)
  • git status(Узнать статус репозитория)
  • git add path to file(добавить новый файл под контроль ГИТА)
  • git commit -am "Здесь комментарий об всех изменениях"(Добавляем новую версию программы)
